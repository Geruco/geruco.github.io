Développeur

Junior

J'ai commencé la programmation à 17 ans. J'ai toujours été passioné par l'informatique, puis un jour l'envie de créer mes propres logiciels et sites web m'est apparu. Mon tout premier travail a été de créer un site pour mon club de sport, mais je ne l'ai jamais proposé, il ne me plaisait pas suffisamment. Ce n'était que ma première tentative de site web. J'ai par la suite commencé divers projet personnels comme des sites internets ou des applications mobiles afin de pratiquer et apprendre par moi-même l'utilisation de certaines technologies, mais aucun d'entre eux n'ont vu le jour. Aujourd'hui, je suis en train de développer avec l'aide d'un ami un site sérieux que je compte bien déployer. Ce projet a pour but d'offrir une documentation précise sur des mods d'un jeu vidéo.

Le développement me plait car il me pousse à rechercher toujours plus de rigueur dans mon travail. J'ai commencé ma formation officiellement en 2017 avec un BTS SIO option Slam avant de poursuivre ces études au Canada. Après avoir souhaité me spécialiser dans la conception de logiciel, je me dirige désormais majoritairement vers un poste de développeur Web back-end. Bien que je dispose également de Compétences liées au front-end, ce domaine m'intéresse moins.

Langages

HTML

HTML

Je crois que tout projet web commence avec un bon contenu, organisé et structuré avec HTML. Cela aide à mettre l'accent sur le contenu. Sans cela, tu risques de perdre de vue pourquoi les utilisateurs accèdent réellement à ton service. Avant toute conception, j'écris toujours du HTML solide.

CSS 3

CSS 3

Le CSS est le maquillage du Web, sans lui tout se ressemblerai sans aucune âme. Cela permet d'afficher des idées, sa philosophie.

Le CSS est la poudre aux yeux mais reste cependant très important, il permet de diférencier un bon site d'un site parfait.

Javascript

Javascript

Mon expérience avec JavaScript est assez grande étant un composant principal de tout projet web. J'aime ce langage pour sa simplicité et son vaste ensemble de possibilités malgré cette simplicité.

MySQL / Oracle

MySQL / Oracle

MySQL et Oracle me permettent d'automatiser ma base de données afin de rendre un projet le plus fonctionnel possible. Il est important que l'utilisateur ne ressent pas une latence durant les traitements des données.
Bien que j'utilise majoritairement MySQL lors de mes projets personnels, j'ai été forcé d'apprendre et utiliser Oracle lors de certains cours, m'offrant une première expérience avec cette technologie.

PHP

PHP

Le php, un langage serveur que j'ai utilisé durant toute la durée de mon BTS afin de produire mon projet de fin d'année.

Python

Python

Le python est un autre langage serveur que j'apprécie beaucoup. Il me plait car il oblige une syntaxe claire car les retours chariots ont toutes leurs importances.
J'ai utilisé d'abord ce langage dans des cours d'algorithmies, mais suite à un cours de technologie web avancées, il m'a été de développé un site en créant un serveur codé en python. J'ai beaucoup apprécié la bibliothèque Python-Flask et c'est désormais mon outil préféré pour créer mes sites internets.

C++

C++

C++ est un langage que j'ai uniquement utilisé pour du développement logiciel de petite envergure. Je connais les bases, mais je n'ai jamais expérimenté un réel développement dans ce langage.

Java / Android Studio

Java / Android Studio

Je dispose d'un peu plus d'expérience dans le Java qu'en C++ par son utilisation dans le développement d'application avec Android Studio. J'ai pu travaillé sur deux applications mobiles de petites envergures en utilisant Android Studio et me suis servi à une seule occasion du Java pour développer une application web.

Bootstrap

Materialize

jQuery

React Native

React Native

React est un framework d'application mobile permettant de simplifier la création d'application mobile multi-plateformes.

Python-Flask

Python-Flask

Python-Flask est une librairie contenant divers outils pour la création d'un serveur web en python. Depuis que j'ai appris son existence, il s'agit du ma manière préférée de développer un serveur web que j'ai utilisé à plusieurs reprises. Cette librairie est souvent combinée à la librairie sqlAlchemy.

sqlAlchemy

sqlAlchemy

sqlAlchemy est une librairie Python permettant de modéliser une base de données en différents objets python. C'est un outil simple et efficace pour se connecter et effectuer des actions sur sa base de données dans une application web. Bien que je trouvais son utilisation difficile au début, j'apprécie de plus en plus cette librairie avec le temps.

Création d'un projet personnel de site internet.

Résumé du projet :

Ce projet consiste à créer un site internet regroupant des articles qui contiennent des informations détaillées à propos de différents mods du jeu Divinity : Original Sin 2. Les articles seront rédigés par des amis et moi-même. Ce site permettra également aux utilisateurs de créer un compte pour commenter et intéragir avec les articles, ainsi qu'utiliser d'autres fonctions qui seront développées ultérieurement.
La création de ce projet est effectuée par un ami s'occupant principalement du front-end, et moi-même m'occupant du back-end. Technologiquement parlant, le site utilise un serveur en python utilisant les librairies Flask et sqlAlchemy et une base de données MySQL.
