Apprenti

Developpeur

J'ai commencé la programmation à 17 ans. J'ai toujours été passioné par l'informatique, puis un jour l'envie de créer mes propres logiciels et sites web m'est apparu. Mon tout premier travail a été de créer un site pour mon club de sport, mais je ne l'ai jamais proposé, je n'en étais pas assez fière. Ce n'était que ma première tentative de site web.

Le développement me plait car il me pousse à rechercher toujours plus de rigueur dans mon travail. J'ai commencé ma formation officiellement en 2017 avec un BTS SIO option Slam avant de poursuivre ces études au Canada. Apres avoir souhaité me spécialiser dans la conception de logiciel, je me dirige désormais majoritairement vers un poste de développeur Web back-end.

Langages

HTML

HTML

Je crois que tout projet web commence avec un bon contenu, organisé et structuré avec HTML. Cela aide à mettre l'accent sur le contenu. Sans cela, tu risques de perdre de vue pourquoi les utilisateurs accèdent réellement à ton service.

Avant toute conception, j'écris toujours du HTML solide.

CSS 3

CSS 3

Le CSS est le maquillage du Web, sans lui tout se ressemblerai sans aucune âme. Cela permet d'afficher des idées, sa philosophie.

Le CSS est la poudre aux yeux mais reste cependant très important, il permet de diférencier un bon site d'un site parfait.

Javascript

Javascript

J'utilise Javascript pour les animations des éléments HTML ou remplacer dynamiquement du contenu.

MySQL

MySQL

MySQL me permet d'automatiser ma base de données afin de rendre un projet le plus fonctionnel possible. Il est important que l'utilisateur ne ressent pas une latence durant les traitements des données.

PHP

PHP

Le PHP.. Sans ce langage magique, tous les sites web seraient statiques et beaucoup moins complet.

Python

Le python n'est pas un langage Web, mais un langage de programmation interprété orienté objet. Il me plait car il permet une syntaxe claire obligatoire car les retours chariots ont toutes leurs importances.

Autres

Atom

Atom

Un éditeur de code permettant de facilement effectuer les commandes Git pour sauvegarder ou récupérer nos dossiers.

Git

Git

Un site internet permettant de sauvegarder les fichiers de code facilement en ligne. Seul bémol, si le service est gratuit, le code est publique.

Brackets

Brackets

Un autre éditeur de code, sûrement celui que j'utilise le plus, ayant un panel de langage supporté très grand, et dispose d'une fonction de visualisation en direct pour les pages Web, ce qui permet d'appliquer le CSS comme on le souhaite.

Photoshop

Photoshop

Photoshop est un logiciel de retouche, de traitement et de dessin assisté par ordinateur, lancé en 1990 sur MacOS puis en 1992 sur Windows.

Blender

Blender

Blender est un logiciel libre et gratuit de modélisation, d’animation et de rendu en 3D.
Il propose des fonctions avancées de modélisation, d’animation 3D, et de rendu. Il gère aussi le montage vidéo non linéaire, la composition, la création nodale de matériaux, la création d’applications 3D interactives ou de jeux vidéo grâce à son moteur de jeu intégré (le Blender Game Engine), ainsi que diverses simulations physiques telles que les particules, les corps rigides, les corps souples et les fluides.

InDesign

InDesign

Un logiciel Adobe permettant de créer le design de mes pages Internets avant de les appliquer en CSS.
